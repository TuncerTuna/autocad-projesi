# AutoCAD Projesi

Bu repository, AutoCAD kullanılarak oluşturduğum teknik çizim projesini içermektedir.

## İçerik
- **drawings/**: DWG formatındaki orijinal AutoCAD çizimi  
- **exports/**: PDF veya görsel formatında dışa aktarılan dosyalar (opsiyonel)

## Hakkında
- Yazılım: AutoCAD
- Dosya türü: .dwg  
- Proje sahibi: Tuncer Tuna Arkun

Bu proje tamamen bana ait olup eğitim, portföy ve referans amaçlı paylaşılmıştır.
